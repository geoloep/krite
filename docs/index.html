krite

Krite is a collection of tools for creating web-based map applications. It mostly consists of wrappers around Leaflet and Ractive.js. Leaflet is used for creating the map component and Ractive.js for rendering the dom of the various side-components. The goal of krite is to create a consistent and pluggable api for interfacing between different kinds of data sources.

As of now tests are still lacking.
